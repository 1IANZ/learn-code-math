# 行列式

## 概念

行列式是一个标量值,用来描述一个方阵(n \* n Matrix)的某些重要性质
行列式是一个标量值,用来描述一个方阵(n \* n Matrix)的某些重要性质。它可以看作是线性变换对“体积”所造成的影响。

### 记法

矩阵 `A` 的行列式记作 `det(A)` 或 `|A|`。

例如，对于一个3x3矩阵:

```
A = | a b c |
    | d e f |
    | g h i |
```

其行列式写作:

```
|A| = | a b c |
      | d e f |
      | g h i |
```

## 直观定义

对于低阶矩阵，行列式的计算有直观的公式。

### 2阶行列式

```
| a b |
| c d | = ad - bc
```

### 3阶行列式 (对角线法则)

```
| a b c |
| d e f | = aei + bfg + cdh - ceg - bdi - afh
| g h i |
```

> **注意**: 对角线法则只适用于2阶和3阶行列式。对于更高阶的行列式，需要使用更通用的方法。

## 几何意义

行列式最直观的理解是其几何意义。

### 二维空间

在二维平面上，由两个向量 `(a, c)` 和 `(b, d)` 构成的平行四边形的**有向面积**就是这两个向量组成的矩阵的行列式：

```
Area = | a b | = ad - bc
       | c d |
```

- 如果行列式为0，说明两个向量共线，平行四边形退化为一条线。
- 行列式的符号（正或负）表示了向量的相对方向。

### 三维空间

在三维空间中，由三个向量构成的平行六面体的**有向体积**就是由这三个向量作为列（或行）组成的3x3矩阵的行列式。

- 如果行列式为0，说明这三个向量共面，平行六面体退化为一个平面图形，体积为0。

## 行列式的性质

行列式有许多重要的性质，这些性质对于简化计算和理论推导都至关重要。

1.  **转置不变性**: `det(A) = det(A^T)`。
2.  **线性性质**:
    - 如果行列式的某一行（或列）的每个元素都乘以一个常数 `k`，则行列式的值也乘以 `k`。
    - 如果行列式的某一行（或列）的元素是两数之和，则该行列式可以拆分为两个行列式之和。
3.  **两行（列）互换**: 互换行列式的任意两行（或列），行列式的值变号。
4.  **两行（列）相同或成比例**: 如果行列式有两行（或列）的元素完全相同或成比例，则行列式的值为0。
5.  **行（列）的倍加**: 将某一行（或列）的 `k` 倍加到另一行（或列）上，行列式的值不变。这是计算行列式时最常用的技巧。
6.  **乘法定理**: `det(AB) = det(A)det(B)`。
7.  **三角矩阵**: 上三角或下三角矩阵的行列式等于其对角线上所有元素的乘积。

## 行列式的计算

对于高于3阶的行列式，通常使用**拉普拉斯展开**（按行或按列展开）或者高斯消元法。

### 代数余子式

- **余子式 (Minor)**: 对于n阶行列式 `A`，去掉第 `i` 行和第 `j` 列的元素后，剩下的 `n-1` 阶行列式称为元素 `a_ij` 的余子式，记作 `M_ij`。
- **代数余子式 (Cofactor)**: 元素 `a_ij` 的代数余子式定义为 `C_ij = (-1)^(i+j) * M_ij`。

### 拉普拉斯展开

行列式的值等于其任意一行（或一列）的各元素与其对应的代数余子式的乘积之和。

- **按第 `i` 行展开**: `det(A) = a_i1 * C_i1 + a_i2 * C_i2 + ... + a_in * C_in`
- **按第 `j` 列展开**: `det(A) = a_1j * C_1j + a_2j * C_2j + ... + a_nj * C_nj`

通过选择含有最多0的行或列进行展开。
